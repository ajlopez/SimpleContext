# SimpleContext

Associate arbitrary data with a context (simple plain JavaScript object).

## Installation

Via npm on Node:

```
npm install simplecontext
```


## Usage

Reference in your program:

```js
var sf = require('simplecontext');
```

## Development

```
git clone git://github.com/ajlopez/SimpleContext.git
cd SimpleContext
npm install
npm test
```
